# Personal Website 👋

## Development setup 🤓

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [NVM](https://github.com/nvm-sh/nvm) (recommended)

### Get started

```sh
git clone git@github.com:henrikfricke/personal-website.git
cd personal-website

# Install dependencies
npm install

# Start 11ty
npm start
```

Go to http://localhost:8080.
